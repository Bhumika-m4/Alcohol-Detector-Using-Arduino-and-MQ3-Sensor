# Alcohol Detector Using Arduino and MQ3 Sensor

## Project Overview
This project presents an Alcohol Detector System developed using an Arduino UNO and MQ3 Alcohol Sensor. The system is designed to detect alcohol vapors in the surrounding environment and generate alerts whenever the detected alcohol concentration exceeds a predefined threshold. The primary objective is to improve safety by preventing alcohol-related accidents and providing a simple, cost-effective monitoring solution.

## Features
- Real-time alcohol detection using MQ3 sensor
- Arduino-based processing and control
- LED indication for visual alerts
- Buzzer for audible alerts
- Easy implementation and low cost
- Suitable for vehicle safety and industrial applications

## Hardware Components
- Arduino UNO
- MQ3 Alcohol Sensor
- LED
- Buzzer
- 220 Ohm Resistor
- Breadboard
- Jumper Wires

## Software
- Arduino IDE
- Embedded C/C++

## Applications
- Drunk driving prevention
- Industrial safety monitoring
- Breath analyzer systems
- Public safety monitoring
